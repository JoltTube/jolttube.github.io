# Security Policy

## Official channels

The only official JoltTube properties are:

- Website: https://jolttube.github.io
- Downloads: https://jolttube.github.io/download
- YouTube: https://www.youtube.com/@FrizzBlox
- Discord: https://discord.gg/379XK7NDY3

Anything else claiming to be JoltTube is not affiliated with this project. Only
download JoltTube from the official links above.

## Reporting a vulnerability

If you find a security issue with the website or the app, please report it
privately through the Discord above rather than opening it publicly, so it can
be fixed before it is disclosed.
